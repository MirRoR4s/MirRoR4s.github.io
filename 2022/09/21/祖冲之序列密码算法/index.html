LFSR初始化模式

初始化模式接收一个31比特字u。u是由非线性函数F的32比特输出W通过舍弃最低位比特得到。

1、注意到在初始化模式中反馈函数一共只使用了16个寄存器中的5个，分别是

$s_{15},s_{13},s_{10},s_4,s_{0}$。但这五个寄存器前面的乘数目前不知道是如何选择的。$2^{15}s_{15}$ 表示$s_{15}$ 左移15位，其余同理。（这里pdf没有明确表明是否这个表示乘法）由此可见，祖冲之序列密码算法的LFSR并不是寻常的定义在GF(2)上的LFSR。

下方代码是不够优秀的，没有实现文档所给的快速模乘算法。

def LFSRWithInitialisationMode(self, u):
    """
    初始化模式下，LFSR接收一个31比特字u，u是由非线性函数F的32比特输出W通过舍弃最低比特位得到
    :param u: 一个31比特字
    :return:
    """

    v = ((self.LFSR_S[15] << 15) + (self.LFSR_S[13] << 17) + (self.LFSR_S[10] << 21) + (self.LFSR_S[4] << 20) + (
                self.LFSR_S[0] + (self.LFSR_S[0] << 8))) % ((2 ** 31) - 1)

    s16 = (u + v) % ((2 ** 31) - 1)

    if s16 == 0:
        s16 = 2 ** 31 - 1
    for i in range(1, 16):
        self.LFSR_S[i - 1] = self.LFSR_S[i]
    self.LFSR_S[15] = s16
ZUC的模加运算

由于模乘涉及到了模加，所以先分析一下模加：

从这里其实也能看出为什么祖冲之序列密码算法要使用31比特的字，应该是考虑到了计算机的实际应用实现。因为现有平台大多都是基于32位/64位处理的，所以使用两个31比特字的加法计算机能够直接存储结果。

这是因为两个31比特字相加，其结果不会超过32比特，不会超过以32位作为字长的计算机的存储范围。

当然，即使某数超过了计算机字长使得计算机无法直接存储，我们也可以在这台计算机上面存储这个整数，只不过不能使用一个计算机字来存储就是了。如果学习过数论及其应用的话，就知道可以使用中国剩余定理来帮助我们解决这类问题。

现在回到算法分析上来，第一步是直接计算a+b的和并赋值给c，这没什么好说的。

重点关注的是第二步，现在假设c是一个32比特的整数。

c&0x7fffffff 即是将c的最高位比特置0

c=(c&0x7fffffff+(c>>31))

那么第二步实际上所做的操作就是将c循环左移一位。

其实从这里文档也教会我们如何进行循环移位的操作，我在模乘的时候用python实现了这个循环移位。

def ModularAddition(self,a,b):
    """

    :param a:
    :param b:
    :return:
    """
    c = a + b
    c = (c&0x7fffffff) + (c >> 31)
    return c

class CyclicShift:
    """
    循环移位类，负责对传入的整数进行循环移位
    """

    def __init__(self, num: int,bit_length:int):
        """

        :param num: 待循环移位的整数
        """
        self.num = num
        self.length = bit_length

    def rotateLeft(self, bit) -> int:
        """
        循环左移方法
        思路是先将num左移bit位 = n1，在将num右移(n-bit)位 = n2，n是传入的的比特位数。最后移位的结果 res = n1 + n2
        :param bit: 左移的bit数量
        :return: 左移后的值，整数
        """
        bit = bit % self.length
        n1 = (self.num << bit) & ((2 ** self.length) - 1)

        n2 = (self.num >> (self.length - bit))
        return (n1 + n2) & ((2 ** self.length) - 1)

    # def rotateRight(self,bit) -> int :

所以现在我们就将a循环左移15位加上a循环左移0位再加上a循环左移0位，注意加法是模 $2^{31}-1$

比特重组 BR

比特重组就是从LFSR的寄存器单元抽取128比特组成4个32比特的字。即BR函数输出4个32比特的字 $X_0,X_1,X_2,X_3$

def BitReconstruction(self):
       """
       比特重组从LFSR的寄存器单元抽取128比特组成4个32比特字X0,X1,X2,X3
       :return:
       """
       X0 = self.BitLink_H(31, self.LFSR_S[15], self.LFSR_S[14])
     

       X1 = self.BitLink_L(31, self.LFSR_S[11], self.LFSR_S[9])

       X2 = self.BitLink_L(31, self.LFSR_S[7], self.LFSR_S[5])
       X3 = self.BitLink_L(31, self.LFSR_S[2], self.LFSR_S[0])

       return [X0, X1, X2, X3]
比特链接的实现：
def BitLink_H(self,n,a, b):
    # assert a.bit_length()<=31
    ans = ((a >> (n - 16)) & (2 ** 16 - 1)) << 16 ^ (b & 0x0000ffff)
    return ans


def BitLink_L(self,n,a,b):
    ans = ((a & 0x0000ffff) << 16) ^ ((b >> (n-16)) & 0x0000ffff)
    return ans

    def __init__(self, key: str, iv: str):
        """
        构造函数实现密钥装入过程，将k和iv扩展为16个31比特字作为LFSR寄存器单元变量的初始状态
        :param key: 128比特的二进制字符串
        :param iv:  128比特的二进制字符串
        """
        assert len(key) == 128 and len(iv) == 128
        """
        初始化阶段
        """
        for i in range(16):
            self.LFSR_S[i] = int(key[8 * i:8 * (i + 1)] + self.D[i] + iv[8 * i:8 * (i + 1)], 2)
        self.R1, self.R2 = 0, 0
        print('LFSR初始状态如下：')
        for i in range(16):
            print("S{}=".format(i), hex(self.LFSR_S[i])[2:])

        for i in range(32):
            X = self.BitReconstruction()
            # print(X)
            X0, X1, X2 = X[0], X[1], X[2]

            W = self.F(X0, X1, X2)
            self.LFSRWithInitialisationMode((W >> 1))

        """
        工作阶段
        """
        print('-----------------------')
        print('初始化之后的LFSR状态如下：')
        for i in range(16):
            print("S{}=".format(i),hex(self.LFSR_S[i])[2:])
        print('-'*20)
        print('初始化之后有限状态机内部状态：',hex(self.R1)[2:],hex(self.R2)[2:])
        X = self.BitReconstruction()
        X0, X1, X2 = X[0], X[1], X[2]
        self.F(X0, X1, X2)  # 舍弃F的输出W
        self.LFSRWithWorkMode()

    def BitLink_H(self, n, a, b):
        # assert a.bit_length()<=31
        ans = ((a >> (n - 16)) & (2 ** 16 - 1)) << 16 ^ (b & 0x0000ffff)
        return ans

    def BitLink_L(self, n, a, b):
        ans = ((a & 0x0000ffff) << 16) ^ ((b >> (n - 16)) & 0x0000ffff)
        return ans

    def ModularAddition(self, a, b):
        """

        :param a:
        :param b:
        :return:
        """
        c = a + b
        c = (c & 0x7fffffff) + (c >> 31)
        return c

    def ModularMultiplication(self, a, b):
        """

        :param a:
        :param b:
        :return:
        """
        tmp = bin(b)[2:][::-1]
        tmp = [i for i in range(len(tmp)) if int(tmp[i]) == 1]
        print(tmp)
        ans = 0
        for i in tmp:
            ans = self.ModularAddition(ans, CyclicShift(a).rotateLeft(i))
        return ans

    def OutputKey(self):
        """
        该函数负责输出一个32比特的密钥字Z
        :return:
        """
        X = self.BitReconstruction()
        X0, X1, X2, X3 = X[0], X[1], X[2], X[3]
        Z = self.F(X0, X1, X2) ^ X3
        self.LFSRWithWorkMode()
        return Z

    def LFSRWithInitialisationMode(self, u):
        """
        初始化模式下，LFSR接收一个31比特字u，u是由非线性函数F的32比特输出W通过舍弃最低比特位得到
        :param u: 一个31比特字
        :return:
        """

        v = ((self.LFSR_S[15] << 15) + (self.LFSR_S[13] << 17) + (self.LFSR_S[10] << 21) + (self.LFSR_S[4] << 20) + (
                    self.LFSR_S[0] + (self.LFSR_S[0] << 8))) % ((2 ** 31) - 1)

        s16 = (u + v) % ((2 ** 31) - 1)

        if s16 == 0:
            s16 = 2 ** 31 - 1
        for i in range(1, 16):
            self.LFSR_S[i - 1] = self.LFSR_S[i]
        self.LFSR_S[15] = s16

    def LFSRWithWorkMode(self):
        """
        工作模式下LFSR不接收任何输入
        :return:
        """
        s16 = ((self.LFSR_S[15] << 15) + (self.LFSR_S[13] << 17) + (self.LFSR_S[10] << 21) + (self.LFSR_S[4] << 20) + (
                    self.LFSR_S[0] + (self.LFSR_S[0] << 8))) % ((2 ** 31) - 1)
        if s16 == 0:
            s16 = 2 ** 31 - 1
        for i in range(1, 16):
            self.LFSR_S[i - 1] = self.LFSR_S[i]
        self.LFSR_S[15] = s16

    def BitReconstruction(self):
        """
        比特重组从LFSR的寄存器单元抽取128比特组成4个32比特字X0,X1,X2,X3
        :return:
        """
        X0 = self.BitLink_H(31, self.LFSR_S[15], self.LFSR_S[14])
        verify = (bin(self.LFSR_S[15])[2:].zfill(31))[:16] + (bin(self.LFSR_S[14])[2:].zfill(31))[-16:]
        assert bin(X0)[2:].zfill(32) == verify

        X1 = self.BitLink_L(31, self.LFSR_S[11], self.LFSR_S[9])
        verify1 = (bin(self.LFSR_S[11])[2:].zfill(31))[-16:] + (bin(self.LFSR_S[9])[2:].zfill(31))[:16]
        assert len(verify1) == 32
        assert bin(X1)[2:].zfill(32) == verify1
        X2 = self.BitLink_L(31, self.LFSR_S[7], self.LFSR_S[5])
        X3 = self.BitLink_L(31, self.LFSR_S[2], self.LFSR_S[0])

        return [X0, X1, X2, X3]

    def L1(self, X: int) -> int:
        """

        :param X: 一个32比特的整数
        :return: X进行线性变换后的值
        """
        x = CyclicShift(X,32)
        return X ^ (x.rotateLeft(2)) ^ (x.rotateLeft(10)) ^ (x.rotateLeft(18)) ^ (x.rotateLeft(24))

    def L2(self, X: int) -> int:
        """
        同L1，但所作的线性变换不同
        :param X:
        :return:
        """
        x = CyclicShift(X,32)
        return X ^ (x.rotateLeft(8)) ^ (x.rotateLeft(14)) ^ (x.rotateLeft(22)) ^ (x.rotateLeft(30))

    def S(self, X) -> int:
        """

        :return:
        """

        x = bin(X)[2:].zfill(32)
        x0, x1, x2, x3 = x[:8], x[8:16], x[16:24], x[24:32]

        y0 = self.ZUC_S0[(16 * int(x0[:4], 2)) + int(x0[4:], 2)]

        y1 = self.ZUC_S1[(16 * int(x1[:4], 2)) + int(x1[4:], 2)]

        y2 = self.ZUC_S0[(16 * int(x2[:4], 2)) + int(x2[4:], 2)]

        y3 = self.ZUC_S1[(16 * int(x3[:4], 2)) + int(x3[4:], 2)]

        Y = (((y0 << 24)) ^ (y1 << 16) ^ (y2 << 8) ^ y3) & 0xffffffff
        assert (bin(y0)[2:].zfill(8) + bin(y1)[2:].zfill(8) + bin(y2)[2:].zfill(8) + bin(y3)[2:].zfill(8)) == bin(Y)[2:].zfill( 32)
        return Y

    def F(self, X0, X1, X2):

        """

        :param X0: 32比特字X0
        :param X1: 32比特子X1
        :param X2: 32比特子X2
        :return: 一个32比特字W
        """

        m = 2**32
        W = ((X0 ^ self.R1) + self.R2) % m
        W1 = (self.R1 + X1) % m
        W2 = self.R2 ^ X2
        t1 = self.BitLink_L(32, W1, W2)
        assert bin(W1)[2:].zfill(32)[-16:] + bin(W2)[2:].zfill(32)[:16] == bin(t1)[2:].zfill(32)
        t2 = self.BitLink_L(32, W2, W1)


        self.R1 = self.S(self.L1(t1))

        self.R2 = self.S(self.L2(t2))

        # print(hex(self.R2))

        return W

总结

祖冲之是一个典型的序列（流）密码，重点关注其密钥流生成算法即可。通过python基本实现了该密码算法，同时再编写代码的过程中都尽量使用位运算，而不是将整数转为字符串进行处理。同时要注重细节，很多密码算法具有雪崩效应，这使得我们在编程中出现的错误也会被放大，只要有一个比特或者字节因为编程错误发生了误差，那么就得不到加密的结果。我就是在L1函数的实现上出现了问题导致卡了1天的时间。
