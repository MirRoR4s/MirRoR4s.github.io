crypto092

题目说了是170号段首批放号的联通号码，百度查到相关信息，然后咱们国家代码是86，未知的哈希值是13位的纯数字，根据已有信息直接爆破。

import hashlib
import string
import itertools
from tqdm import tqdm
alp = string.digits
a1 = '86170'
tar = 'c22a563acc2a587afbfaaaa6d67bc6e628872b00bd7e998873881f7c6fdc62fc'
for k in range(7,20):
    a = a1 + str(k)
    for i in tqdm(itertools.product(alp, repeat=7)):
        tmp = ''.join(i)
        num = a + tmp
        hash = hashlib.sha256(num.encode()).hexdigest()
        if hash == tar:
            print(num)
            break

首先需要注意的是，根据题目源代码，当前轮的grasshopper跟当前轮的密钥是一致的。其次我们可以发现第一组密文为$102k0k1k2k3k4$，而第二组密文为$1192407267456k0^5k1^4k2^3k3^2k4$。不难发现第二组密文即当前轮grasshopper4=上一轮的grasshopper3乘以上一组密文。这里的轮数指的是对每一个加密的字母所进行的5次关于grasshoper的迭代，也就是加密过程中的内层循环。

$C_i = C_{i-1}\cdot g_3$

crypto162

密钥是多个递归函数之和，由于递归太多次了，显然我们不能直接运行源代码求出这个和。阅读cal函数具体实现就不难发现存在一个递推方程

cal(i,cof) = cof[2]*cal(i-3,cof)+cof[1]*cal(i-2,cof)+cof[0]*cal(i-1,cof)，其中初始状态是1,2,3

所以我们直接把递归求和变为递推方程求和即可求出密钥。为了加快运算速度，注意到密钥最后只取了这个大数的倒数两千位和倒数一千位之间的一千位，所以求和的时候可以进行一个取模操作；值得注意的是这个加密代码应该是用py2写的，解密我们最好也用py2。
