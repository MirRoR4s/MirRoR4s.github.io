3.2.7 Ysoserial

Ysoserial 是一款开源的 Java反序列化测试工具，内部集成有多种利用链，可以快速生成用于攻击的代码，也可以将新公开的反序列化漏洞利用方式自行加入 Ysoserial 中。

3.2.8 Marshalsec

Marshalsec 是一款 Java 反序列化测试工具，不仅可以生成各类反序列化利用链，还可以快速启动恶意的 RMI 服务。

3.2.9 MySQL 监视工具
