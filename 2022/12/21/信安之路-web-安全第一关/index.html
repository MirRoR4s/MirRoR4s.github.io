修改Nginx默认配置#

黑客经常抓包分析我们服务器的响应头，而很多服务器并没有对返回的响应头做一个过滤，这会泄露很多我们服务器的信息，比如服务器类型、版本号等。所以现在我们修改一下 nginx 的配置文件，将一些不安全的HTTP响应头删掉。

# 控制缓冲区大小
client_body_buffer_size 1K;
client_header_buffer_size 1k;
client_max_body_size 1k;
large_client_header_buffers 2 1k;



# 清除不安全的HTTP响应头
more_clear_headers "X-Powered-By";
more_clear_headers "Server";
more_clear_headers "ETag";
more_clear_headers "Connection";
more_clear_headers "Date";
more_clear_headers "Accept-Ranges";
more_clear_headers "Last-Modified";

# 禁止不安全的 HTTP 方法
if($request_method !~ ^(GET|HEAD|POST)$){
    retrun 444;
}
UFW#

ufw是一个主机端的iptables类防火墙配置工具，比较容易上手。一般桌面应用使用ufw已经可以满足要求了。我们可以通过配置防火墙，允许或者禁止一些特定端口的进出，比如不允许外来的IP连接我们，只允许我们连接别人等。
