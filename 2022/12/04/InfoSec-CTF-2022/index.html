已知flag的前五位是 flag{，可以利用已知明文攻击获得密钥的前五个字节。然后我们爆破密钥的第六个字节（共有256种可能），这样就得到了密钥的前六个字节。然后我们根据明文的第六个到第十一个字节是和密钥的前六个字节异或的特点可以恢复明文的六到十一个字节。恢复这部分明文之后同理就可以拿到密钥的六到十一字节。

QR code

题目描述

Crypto

Alice decided to send Bob a QR code. To do this, she generated a key using a physical random number generator, the length of which did not exceed 50,000 bytes. Next, she performed a byte-by-byte xor operation of QR code with a repeating key. Can you recover the QR code?

解题步骤

  1. 注意到加密的QR code 文件的格式是 .bmp，如果你以这种格式保存寻常的黑白 QR code并在一个十六进制编辑器种解释其内容，你就能够发现此文件种大部分字节要么是全1（白色像素点），要么是全0（黑色像素点）。
  2. 可以假设此加密文件的末尾仅包含 字节 0xFF（they form the bottom of the white frame that delimits the QR code）。给定文件的大小超过了 $410^6$，此文件的末尾的白色像素点的数量一定大于 $510^4$ 字节（最大的密钥长度）
  3. 我们可以假设异或运算是一组一组进行的，并且分组大小等于密钥长度。这就意味着加密文件的末尾存在一些分组，这些分组是由密钥和 0xFF异或而来。
  4. 因为最后一个分组可能不是完整的（如果密钥的长度不是文件长度的整数倍），所以我们要选择倒数第二个分组。我们发现最后一个分组的长度是文件的长度除以预期的密钥长度的余数。倒数第二个分组以一个字节开始，从文件末尾到它的距离等于密钥的预期长度加上最后一个块的长度。
  5. 为了了解我们是否在使用正确的密钥长度，我们可以先恢复密钥的前两个字节（通过将倒数第二个块的前两个字节和 两个 0xFF 异或）。如果我们将密钥的前两个字节和加密文件的前两个字节相异或，那么结果应该是 [0x42,0x4D]（bmp文件头）。
  6. 找到所有使得加密文件的前两个字节能够被正确解密的所有密钥长度。对于得到的值，通过将加密文件的倒数第二个块和 0xFF 异或 来重构出完整密钥。
  7. 使用找到的密钥解密文件。我们可以手动地查看所有恢复的文件并找到那个可以正常显示的 QR code图片。
  8. 扫描解密后的QR code 获取 flag

Love and Matrices

题目描述

Will matrices help to encrypt the flag properly?

解题步骤

这其实是一个超级简单的问题，仅仅需要一点点线性代数的知识就可以做出来。设 三个非奇异矩阵 X，Y，Z，其中 Z的行列式值为1且 Y = X * Z 。根据相关知识可以知道 Z的行列式值等于 X的行列式值乘以 Y 的行列式值。想具体的了解可以看这篇文章：矩阵乘积的行列式等于行列式的乘积 |AB|=|A||B| - 知乎 (zhihu.com)

题目中恰好就生成了一个行列式值为 1 的 32 行 32 列的矩阵 Z，相关函数的定义在sage文档里写了：

https://doc.sagemath.org/html/en/reference/matrices/sage/matrix/special.html#sage.matrix.special.random_matrix

题目告诉我们如果 flag 的当前比特是 1，就让 X 乘以 Z 并赋值给 Y。根据上述知识可以知道当 X 和 Y 的行列式值相等的时候，我们就可以认定 X 乘以了 Z 并认为flag 当前比特是 1。那如果 X 和 Y的行列式值在不乘以Z的也相等呢？这种情况发生的概率很低，因为随机生成两个32行32列的矩阵，其行列式值一般都是不等的。

evil_fibonacci

题目描述

解题步骤

密钥是斐波那契数列的第N项。希望找到一个在具有p个元素的有限域上的多项式 $f_n$，使得 $f_n$ 和斐波那契数列的项具有某些关系。可以如下定义一个多项式 ：$f_n(x) = a_n + b_n * x$，也就是 $x^{n+1}$ 除以 $x^2-x-1$ 的余数，系数运算模p。我们发现：

$f_0(x) = 0 + 1 * x$

$f_1(x) = 1 + 1 * x$

$f_2(x) = 1 + 2 * x$

$f_3(x) = 2 + 3 * x$

可以发现当 $n>=1$，我们有如下公式：

$f_n(x)=x*f_{n-1}(x),mod,(x^2-x-1)=b_{n-1}+(a_{n-1}+b_{n-1},mod,p)*x$

这就表明 $a_n=F_n,b_n=F_{n+1}$ ，$F_i$ 是斐波那契数列的第 i 项。

所以 求 $F_n$ 即是 求 $a_n$，也就是要求 $f_{n}=x^{n+1},mod,(x^2-x-1)$。
