MODULAR MATH

Quadratic Residues

我们已在之前见过了模算数的乘法和除法，但是在模一个整数下的平方根意味着什么呢？

对以下的讨论，让我们以模 p=29 为例，我们可以让整数 a=11 并计算 $a^2=5\mod\,29$

因为 $a=11,a^2=5$ 我们说 5 的平方根是 11。这感觉很好，但是现在让我们考虑一下 18 的平方根，从以上的分析我们知道我们需要找到一个整数 $a$ 使得 $a^2=18$

你的首要想法也许是 从 $a=1$ 开始一直迭代到 $a=p-1$。在这个讨论中的p不是很大，所以我们可以快速地找到a

试试吧，尝试用代码实现你的想法并看看你找到的整数a是什么。如果你的编写是正确的，那么你将会发现 对 $a\in F_p^*$ 你永远也找不到一个整数 a 使得 $a^2=18$

实际上不是所有 $F_p^$ 中的元素都有一个平方根， $F_p^$中只有一半的元素存在平方根，一半不存在。

我们说整数 $x$ 是一个二次剩余如果存在一个整数 $a$ 使得 $a^2=x\mod\,p$

如果不存在这样的整数，那么该整数是一个二次非剩余。

换句话说， $x$ 是一个二次剩余当我们在模 p 下求得其平方根是可能的时候。

以下的列表存在二次剩余以及二次非剩余，将两个根中较小的那个提交作为flag。

如果 $a^2 =x$ ，那么 $(-a)^2=x$，所以如果 x 某个有限域中的二次剩余，那么整数a的解的个数是2。

Legendre Symbol

在二次剩余中我们学习了求模一个整数的平方根是什么意思，同样我们也看到根并不是一直都存在的。

在之前的案例中 $p=29$ ，即使计算平方根的最简单的方法足够快，但是随着p的增大，该方法变得非常不合理。

幸运地是，我们有一个方法可以验证一个整数是否是二次剩余，该方法由 Legendre发明，只需要采用一个简单的计算即可。在以下我们将假设我们正在对素数p求模。

在思考 Legendre’s 符号之前，让我们先来简短地看看二次剩余和二次非剩余的有趣的性质。

二次剩余 乘以 二次剩余 = 二次剩余
二次剩余 乘以 二次非剩余 = 二次非剩余
二次非剩余 乘以 二次非剩余 = 二次剩余
速记小技巧，可以将 二次剩余用 +1 代替，二次非剩余用 -1 代替，以上三个结果对1 和-1都是相同的

所以秘诀是什么呢？ Legendre Symbol 符号给出了决定一个整数是否是一个模奇数p的二次剩余的高效的方法。

Legendre's Symbol：$(a/p)\equiv\,a^{(p-1)/2}\mod\,p$ 遵循以下三个性质：

(a / p) = 1 if a is a quadratic residue and a ≢ 0 mod p
(a / p) = -1 if a is a quadratic non-residue mod p
(a / p) = 0 if a ≡ 0 mod p

这意味着对于给定的整数 a ，计算 pow(a,(p-1)/2,p) 的值足以决定 a 是否是一个二次剩余

现在开始获取 flag。给定以下 1024 比特的素数和 10 个整数，找到二次剩余并计算其平方根，平方根就是你的flag。对于两个可能的根，提交较大的那一个作为你的答案。

因此 Legendre's symbol 告诉我们整数是一个二次剩余，但是现在我们怎样找到平方根呢？遵循p = 3 mod 4 的素数p 允许我们简单地计算平方根。答案在网络上，但是你可以尝试运用费马小定理自己找出答案吗？

Modular Square Root

在 Legendre Symbol 中我们介绍了一个快速决定一个整数是否是模素数的二次剩余的方法，我们可以更进一步：存在算法可以高效地计算二次剩余的平方根。在实践中最好的一个方法被称为 Tonelli-Shanks 。

除了2以外，所有的素数p都满足以下两个同余式之一： $p\equiv1\,\mod4$ 或者 $p\equiv3\,\mod\,4$ ，因为所有的奇数都满足这两个同余式之一。正如之前的挑战所提示的，在 $p\equiv3\,\mod4$ 的情况下，我们可以由费马小定理直接导出一个计算平方根的非常简单的公式。

但是 $p\equiv1\,\mod\,4$ 的情况还没有解决，我们需要一个更加通用的算法。

在一个具有 $r^2\equiv a\,\mod\,p$ 的同余式中，Tonelli-Shanks 计算 r。

Tonelli-Shanks doesn't work for composite (non-prime) moduli. Finding square roots modulo composites is computationally equivalent to integer factorization.

Tonelli-Shanks 算法对合数的模数无效，找到模为合数的平方根在计算上等价于整数的因子分解。

该算法的主要使用情形是找到 椭圆曲线的坐标，它的运算有点复杂。因此，我们将不会讨论具体的细节。然而 算法的实现很容易找到，Sage 已经内置了该算法。

Chinese Remainder Theorem

中国剩余定理能够给出一个线性同余方程组的唯一解，如果这些模数之间是互素的话。这就意味着，给定一组任意的整数 $a_i$ 以及成对的互素整数 $n_i$，使得以下线性同余式成立：

（成对的互素整数意味着如果我们有一个整数集合 ${n_1,n_2,…,n_i}$，那么从集合中任意选择两个整数其最大公因子都为1）

x ≡ a1 mod n1
x ≡ a2 mod n2
...
x ≡ an mod nn

那么存在在模 $N$ 下存在一个唯一解 $x$， x ≡ a mod N，N = n1 * n2 * ... * nn

from Crypto.Cipher import AES
from Crypto.Util.number import inverse
from Crypto.Util.Padding import pad, unpad
from collections import namedtuple
from random import randint
import hashlib
import os

# Create a simple Point class to represent the affine points.
Point = namedtuple("Point", "x y")

# The point at infinity (origin for the group law).
O = 'Origin'

FLAG = b'crypto{??????????????????????????????}'


def check_point(P: tuple):
    if P == O:
        return True
    else:
        return (P.y**2 - (P.x**3 + a*P.x + b)) % p == 0 and 0 <= P.x < p and 0 <= P.y < p


def point_inverse(P: tuple):
    if P == O:
        return P
    return Point(P.x, -P.y % p)


def point_addition(P: tuple, Q: tuple):
    # based of algo. in ICM
    if P == O:
        return Q
    elif Q == O:
        return P
    elif Q == point_inverse(P):
        return O
    else:
        if P == Q:
            lam = (3*P.x**2 + a)*inverse(2*P.y, p)
            lam %= p
        else:
            lam = (Q.y - P.y) * inverse((Q.x - P.x), p)
            lam %= p
    Rx = (lam**2 - P.x - Q.x) % p
    Ry = (lam*(P.x - Rx) - P.y) % p
    R = Point(Rx, Ry)
    assert check_point(R)
    return R


def double_and_add(P: tuple, n: int):
    # based of algo. in ICM
    Q = P
    R = O
    while n > 0:
        if n % 2 == 1:
            R = point_addition(R, Q)
        Q = point_addition(Q, Q)
        n = n // 2
    assert check_point(R)
    return R


def gen_shared_secret(Q: tuple, n: int):
    # Bob's Public key, my secret int
    S = double_and_add(Q, n)
    return S.x


def encrypt_flag(shared_secret: int):
    # Derive AES key from shared secret
    sha1 = hashlib.sha1()
    sha1.update(str(shared_secret).encode('ascii'))
    key = sha1.digest()[:16]
    # Encrypt flag
    iv = os.urandom(16)
    cipher = AES.new(key, AES.MODE_CBC, iv)
    ciphertext = cipher.encrypt(pad(FLAG, 16))
    # Prepare data to send
    data = {}
    data['iv'] = iv.hex()
    data['encrypted_flag'] = ciphertext.hex()
    return data
