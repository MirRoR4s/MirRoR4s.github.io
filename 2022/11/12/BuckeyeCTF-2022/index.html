fastor 3-傅里叶变换在图片上的简单应用#

题目描述：#

Make the image different yet same!

https://fastfor.chall.pwnoh.io

题目代码：#

分析：#

有一些陌生的函数

shape#

该函数返回的是一个 数组维度的元组

bitwise_xor#

将两个数组中的元素取出并按位异或，返回结果是一个新数组或者是一个标量，在本题显然是一个数组。

sum#

在给定轴上的数组元素之和

fft#

离散的傅里叶变换。fftn是进行n维的离散傅里叶变换。根据下文的std函数的参数该函数应该返回了一个

std#

计算沿指定轴的标准偏差。

所以 check_hash 函数所作的事情就是打开两张图片，一张是我们上传的，一张是题目自带的。之后将这两张图片转为 数组，或许应该叫矩阵

题目的要求如下：

1、两张图片的 shape 要相同，这里应该是说矩阵的维度要相同，那么图片的尺寸应该要一样。

2、两张图片不能完全相同，因为异或后的和不能是0

3、两张图片转为矩阵后进行的n维离散傅里叶变换的标准偏差大于 -1 小于 1

7. Quad prime RSA -连分数RSA#

题目描述#

Ok I learned from my mistake! I’ll just put my twin primes into separate RSA keys…

题目内容#

分析：#

So if $\left|\frac{n_1}{n_2} - \frac{p}{s}\right| < \frac{1}{2s^2}$, then we can definitely find $\frac{p}{s}$ as a convergent of $\frac{n_1}{n_2}$

You can do some algebra to find that $\left|\frac{n_1}{n_2} - \frac{p}{s}\right| = \left|\frac{-2p}{(q + 2) s} \right| < \frac{4s}{2^{23} s^3} < \frac{1}{2s^2}$

Using the fact that $q + 2 > 2^{23} s^2$ and $p < 2s$ for the first inequality
