Introduction#

许多应用程序使用 JSON Web Tokens (JWT) 来允许客户端指示身份，以便在身份验证后进一步进行信息的交换。

一个 JWT token 的结构#

token 由 三部分经过了 base64编码的数据组成：

1、header

2、claims

3、signature

标头和声明都由 JSON 对象标识。标头描述了用于 JWT的加密操作以及一些可选的、额外的 JWT 的性质。
