审计一下发现服务端获取 ip 的过程很简单，就是从客户端的 HTTP 首部中提提取 HTTP_CLIENT_IP、HTTP_X_FORWARDED_FOR 等变量的值，只要这些变量中有一个不为 unknown，那就把相应变量赋值给 ip。这里肯定是我们可控的！

然后还有一个 if 判断 getiptype 是否大于 0，在 config.php 可以看到这个变量默认是0，所以不会进行 if 判断！最后又对 ip 参数作了一个过滤，这个 cms 这么懂安全的吗？

time() 函数返回自 Unix 纪元（January 1 1970 00:00:00 GMT）起的当前时间的秒数。这个是PHP内置的函数，没什么好说的。
