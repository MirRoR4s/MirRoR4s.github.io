1. 安装 XDebug

先去 https://xdebug.org/wizard 这里把你的 phpinfo 的输出复制上去，随后根据推荐指南下载即可

以下是我的指南：

  1. Download php_xdebug-3.1.6-7.3-vc15-nts-x86_64.dll
  2. Move the downloaded file to C:\phpstudy_pro\Extensions\php\php7.3.4nts\ext, and rename it to php_xdebug.dll
  3. Update C:\phpstudy_pro\Extensions\php\php7.3.4nts\php.ini and add the line:
    zend_extension = xdebug
  4. Restart the webserver
