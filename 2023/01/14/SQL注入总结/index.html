SQL注入总结

2023-01-14 (Updated: 2023-01-20)
SQL注入
SQL注入

做过很多个 SQL 注入的题，于是我想总结自己的 SQL 注入知识点。

绕过空格过滤#

一般代替空格的字符串有

  • %09
  • %0a
  • %0b
  • %0d
  • %0c

绕过回显关键字过滤#

有时候题目可能会在回显的时候限制不能出现 flag 等字符，这种情况下我们一般可以采用编码绕过，常见的编码函数有

  • to_base64
  • hex

如果题目还限制了不能出现数字，那我们只能采用盲注了
