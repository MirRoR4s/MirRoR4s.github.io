以上代码做了什么呢？

  1. 首先我们导入了 Flask 类，该类的一个实例将会是我们的 WSGI 应用。
  2. 下一步我们创造了该类的一个实例，第一个参数是应用的模块或包的名字。__name__ 是一个方便的快捷方式，适用于大多数情况。我们需要这个参数来告诉 Flask 去哪里寻找资源，比如 模板或静态文件
  3. 然后我们使用 route() 装饰器告诉 Flask 什么 URL 应触发我们的函数
  4. 函数会返回我们想在用户的浏览器上展示的信息，默认的 content type 是 HTML，所以在 string 中的 HTML 将会被浏览器渲染
